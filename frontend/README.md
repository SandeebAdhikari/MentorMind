## Features

- Group collaboration features (upcoming)

## Tech Stack

- **Frontend:** Next.js, Tailwind CSS, DaisyUI
- **Backend:** Spring Boot, PostgreSQL, JWT Auth
- **AI:** GPT-4 API (optional)
- **Deployment:** Vercel (frontend), Render/Heroku (backend)
